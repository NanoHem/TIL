핵심 쏙쏙 Git - 2주차

머리말
3주차에서는 협업을 위한 작업 관리와 Git 프로젝트 관리-협업 매너, 오픈소스로 정보 탐색하고 함께즐기기, Github로 개발자인 나를 나타내자 포트폴리오 가있다.

내용
PR은 풀리퀘스트이다. 작업내역을 바로 머지하지 않고, 참여하고 있는 프로젝트에 내 작업을 머지해달라고 요청을 하는 것이다. 이걸 응용해서 다른 repo에 PR(fork도 사용)을 하는 것을 알았다. 그리고 최신 commit 고치는 방법인 amend를 사용해보았고 되돌리기인 revert,reset을 사용해 보았다.
임시 보관함인 stash를 사용해보았다. .gitingnore을 사용하여 필요하지만 github와 소스트리에는 불필요한 파일을 올리지 않는 방법을 배웠다. 그 밖에도  오픈소스와 다른 개발자의 sns를 활용하는 법을 배웠다.
